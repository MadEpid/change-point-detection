## Introduction

Change-point methods are useful for detecting parameter instabilities or changes in the moments of the data. The package provides tools for detecting abrupt (i.e., breaks) or smooth change-points in any of the followings: mean, variance, autocovariance and autocorrelation. The methods are based on frequency domain statistics as developed in [Casini and Perron (2021)](https://alessandro-casini.com/wp-content/uploads/2021/03/CASINI_PERRON_Change-Point_Spectrum_SLS.pdf), "Change-Point Analysis of Time Series with Evolutionary Spectra".
